Auto LinkedIn Applier

Auto Jober was a group project I did with three other people that focused on automating job applications.

Running the program creates a GUI prompt that asks for your LinkedIn profile credentials. This helps the program log into LinkedIn and start scraping for "Easy Apply" jobs. The rest of the prompted fields are then used to fill in the required information for these jobs.

This would be an example of how the input fields on LinkedIn would look like after running the program. In this example, the email and number is automatically filed out, and other fields like the resume would also fill out. Unfortunately, unique fields like years of experience would have to be filled out manually.
